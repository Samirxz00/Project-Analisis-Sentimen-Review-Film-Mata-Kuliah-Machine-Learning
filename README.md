## üìñ Description

**Analisis Sentimen Review Film** is a project focused on comparing Natural Language Processing (NLP) methods. It integrates techniques from Machine Learning, Deep Learning, and advanced models like Transformers and LoRA. This application uses the **IMDB Movie Reviews (50K data)** dataset to classify sentiment as either *positive or negative*.
